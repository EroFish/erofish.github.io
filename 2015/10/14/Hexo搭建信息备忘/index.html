设定目录为Hexo

cd d:\Hexo

清理

hexo clean

搭建

hexo g

本地预览（debug）

hexo s

当自己比较胸有成竹的时候,也可以清理搭建预览一步到位:

hexo clean && hexo g && hexo s

上传Github

hexo d

同样,也可以清理搭建上传一步到位:

hexo clean && hexo g && hexo d
